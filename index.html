LiftLog Android App

LiftLog is a simple, easy to use android app to keep track of your weight-lifting progress.

The basics:

  • Exercises
    • Add the Exercises that you will be performing. Example
    • When you first install the app, a few default exercises will be provided.
  • A Session containing multiple Lifts
    • Create a Session and add Lifts to it. Example
    • A Lift represents the lifts performed in that workout (an Exercise performed at some weight times some reps times some sets) e.g. Bench Press @ 225lb x 5 reps x 5 sets

Other features:

  • View History
    A tool to view the complete history of a given exercise.
    • Navigate to it via the tools tab on the main screen. Example
    • Navigate to it via the Lift screen (it will show the history for the currently selected exercise) Example
  • 1-Rep Max Calculator
  • Find Best Lifts
    A tool to search for the best lift you have performed.
    • For example, search for the best Bench Press I have performed for 1 rep.
